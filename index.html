Pronomen Man · Zeitadverb · Konjunktionen
Pronomen man

man = anyone/one/people (3rd person singular conjugation)

Man kann in der Altstadt spazieren gehen.

💡Mann (capital) = a male person. man (lowercase) = indefinite pronoun.
Zeitadverben word order

zuerst → dann → danach → später → zum Schluss

Pos.1 = time adverb → Pos.2 = verb → Pos.3 = subject

Zuerst wollen sie frühstücken, dann machen sie eine Stadttour.

Konjunktion denn / aber

denn = because (answers Warum?) — verb stays pos.2 in next clause

Die Stadt ist toll, denn man kann viel machen.

aber = but · Die Arbeit ist interessant, aber anstrengend.

Imperativ — 3 forms
  • du: Mach! (remove -st, drop du)
  • ihr: Macht! (drop ihr)
  • Sie: Machen Sie! (keep Sie)
  • sein: Sei! / Seid! / Seien Sie!
  • -ten verbs add -e: Warte! Arbeite!
Modal verb word order
  • Modal = position 2, Infinitiv = END
  • Ich muss heute einkaufen gehen.
  • Yes/no: modal first → Kannst du kommen?
  • nicht dürfen = not allowed
Cases quick rule
  • Nominativ: subject / after sein
  • Akkusativ: direct object / nach für
  • Dativ: after location prepositions
  • der→dem · das→dem · die→der · die(pl)→den+n
Wo / Wohin / Woher
  • Wo? → im, in der, beim
  • Wohin? → zum, zur, nach
  • Woher? → vom, aus, von
  • zu Hause · nach Hause · von zu Hause

Perfekt formula
  • haben/sein (pos.2) + Partizip II (end)
  • Regular: ge + stem + t (gemacht)
  • Irregular: ge + stem change + en (getrunken)
  • -ieren: no ge-, just …t (studiert)
  • trennbar: an+ge+zogen = angezogen
  • nicht trennbar: no ge- (bezahlt)
Sein-Verben (not haben!)

  • kommen→gekommen · bleiben→geblieben
  • reisen→gereist · aufstehen→aufgestanden
  • passieren→passiert · ankommen→angekommen
Demonstrativartikel dieser
  • der → dieser (Nom) / diesen (Akk) / diesem (Dat)
  • das → dieses (Nom/Akk) / diesem (Dat)
  • die → diese (Nom/Akk) / dieser (Dat)
  • Welchen Mantel? — Diesen hier.
